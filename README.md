# _Typerwell_ 
Like [Tinkerwell](https://tinkerwell.app) :dizzy: for TypeScript. A VS Code extension.

Under construction :warning:

## Dev installation
`yarn`

## Experiment 1
Put your code in `sample.ts`.
Run `npx ts-node tinker.ts`
Expect result of last statement to be logged
